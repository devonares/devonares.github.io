[[ Twine resources ]]

I make games with twine, and here are resources that have helped me:

  • Tumblr masterlist of various twine resources by idrelle games
      • Basically a compilation of their posts and answers and there are a lot of useful codes, tutorials, and links to templates.
  • Modern developer workflow guide for twine by lazerwalker
      • If you want to develop a twine game using VS Code and automated github pages deployment, which are tools I am more comfortable with working.
  • Useful SugarCube 2 macros by Chapel
      • So many useful stuffs here, like adding custom pronouns to your game, inventory system, speech box, and so much more.
  • Ready to use tweego project folder by manonamora
      • Helpful tweego (twine compiler) project folders template to get started quickly.
